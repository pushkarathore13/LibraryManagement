\# Library Management System



A Library Management System developed using Java EE.



\## Technologies Used



\- Java

\- JSP

\- Servlets

\- JDBC

\- MySQL

\- Maven

\- Apache Tomcat 9

\- Log4j2

\- BCrypt

\- Redis



\## Features



\### Student Module

\- Student Registration

\- Login

\- Forgot Password

\- Change Password

\- Edit Profile

\- Seat Selection

\- Membership Management

\- Seat Change Request



\### Admin Module

\- Admin Login

\- Approve Student Registration

\- Reject Student Registration

\- Search Student

\- View Active Members

\- View Seat Change Requests

\- Approve Seat Changes



\## Database



\- MySQL Database

\- JDBC Connectivity

\- Singleton DB Connection



\## Security



\- BCrypt Password Hashing

\- Session Management

\- Authentication Filters

\- Input Validation using Regex
